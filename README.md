# Lightroom 神经网络一键调色工作站 (Local Workspace)

这是一个融合了大语言模型（LLM）美学特质与神经网络级调色算法的专业摄影后期工作流。它能将您的相机 RAW 文件夹/照片集与 AI 灵感直接桥接，在浏览器中完成直观的色彩、影调、HSL 微调后，一键生成 Lightroom 的 `.XMP` 预设，并提供与本地 Lightroom Classic 客户端同步的开发接口。

---

## ⚡ 极速本地部署指南

##  注意：此应用调用的API目前仅支持Gemini，请先进入网页后在网页下方点击连接端口，下载解压后--在终端--进入文件目录后（在终端）输入"npm install undici",之后输入$env:HTTP_PROXY="http://127.0.0.1:xxxx"\n $env:HTTPS_PROXY="http://127.0.0.1:xxxx"\n npm run dev,这里的xxxx是您的VPN端口，并且将"server.ts"文件中第9行的7897端口更改成您的VPN端口

### 1. 自动一键启动 (推荐 Windows 用户)
如果您是 **Windows** 系统用户，无须手动输入命令，只需简单两步：
1. 双击运行目录下的 **`双击一键本地启动.bat`**。
2. 脚本会**自动检测 Node.js 环境**、**一键下载依赖库**，并在 3 秒内**自动拉起浏览器**进入系统：[http://localhost:3000](http://localhost:3000)。

*(提示：第一次启动时会自动下载第三方依赖，可能需要 15~30 秒，后续双击即可秒级打开服务。)*

---

### 2. 手动启动 (全平台 / macOS / Linux)
如果您使用的是 macOS 或希望手动管理，请打开终端 (Terminal) 执行以下常用步骤：

#### 第一步：安装 Node.js 运行环境
- 确保您的电脑在本地安装了 Node.js （推荐长期支持版 v18 / v20 / v22）。
- 可以在终端输入以下命令查看：
  ```bash
  node -v
  ```

#### 第二步：安装依赖库
定位到解压后的项目根目录下，执行命令：
```bash
npm install
```

#### 第三步：启动本地服务
执行如下命令：
```bash
npm run dev
```
看见控制台输出 `Server running on http://localhost:3000` 后，在浏览器中打开主界面即可：
 👉 [**http://localhost:3000**](http://localhost:3000)

## 🎨 系统主要功能特征

1. **RAW 图像一键检索与智能预览**
   - 支持从本地或拖拽中匹配并载入 RAW 高画质作品队列（在无法直接处理的情况下，会自动装载 4K 画幅的高级演示底片资产）。

2. **多态色彩开发面板**
   - **白平衡与基础影调**：色温(Temp)、色调(Tint)、曝光度(Exposure)、对比度(Contrast)、高光、阴影、黑白位等精细微调。
   - **质感渲染**：清晰度(Clarity)、去薄雾(Dehaze)、鲜艳度、饱和度。
   - **HSL 色彩混色器**：完整集成了摄影中至关重要的 **红、橙、黄、绿、浅绿、蓝、紫、洋红** 8 色色相、饱和度、明度细致控制。

3. **双擎微调模式**
   - **快速预设模式**：一键应用内置的“哥本哈根迷雾(冷色调)”、“黄金时刻胶片(复古)”、“新潮宿霓虹(赛博朋克)”、“经典黑白大都市”等高端相机色调。
   - **专业色彩语言微调**：内置的 AI 神经算法一键色彩风格提取、自定义 Prompt 文学语意生成。

4. **无缝导出与本地 Lightroom 桥接**
   - **导出 XMP**：一键将当前的调色、曲线及 8 色 HSL 参数生成合规的 Lightroom Standard Adobe `.XMP` 预设配置文件，供电脑端双击直接载入。
   - **同步 LR (本地同步桥接)**：一键发送调控信号到本地 18000 端口（可以通过开发工具中的本地进程或 Lightroom API 进行直接联动）。

## 🛠️ 项目主要结构说明

- `index.html` & `src/main.tsx` - 主程序入口。
- `src/App.tsx` - 工作站核心可视化交互框架。
- `src/components/GradingSliders.tsx` - 色彩调理滑块组件（含最新扩展的 HSL 选项卡）。
- `src/components/ImageComparison.tsx` - 镜头原图(Before)与调色后(After)无缝滑动对比幕布。
- `src/utils/filter.ts` - 调色预设定义。
- `server.ts` - 用于管理和监听 3000 端口的主服务器（并在后台执行 Lightroom 客户端同步模拟接口）。
- `双击一键本地启动.bat` - 专为本地非开发者定制的一键式启动脚本。
